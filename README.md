 # 📝 VI Notes – Human Writing Authenticity Platform


## 🚀 Overview
VI Notes is an intelligent writing platform that verifies **human-written content authenticity** by analyzing real-time typing behavior. Instead of analyzing the content itself, the system captures how users write, including typing rhythm, keystroke patterns, and editing behavior.

By leveraging behavioral signals, VI Notes distinguishes between genuine human writing and AI-generated or AI-assisted content. The platform supports academic integrity, enables authorship verification, and ensures privacy by storing only typing metadata, not the actual written content.



## ✨ Core Features

- **Real-Time Writing Editor**  
  Clean and distraction-free interface for seamless writing sessions  

- **User Authentication**  
  Secure login and registration system  

- **Typing Behavior Analysis**  
  Captures key metrics such as:  
  - Keystroke timing  
  - Typing pace  
  - Average key interval  
  - Pause count  
  - Deletions and corrections  
  - Space key usage  

- **Paste Detection System**  
  Detects pasted text and differentiates between typed and AI-assisted content  

- **Authenticity Scoring**  
  Generates authenticity percentage based on typing vs pasted content  

- **Insights Dashboard**  
  Provides detailed analytics on typing behavior after each session  

- **Session Storage**  
  Stores writing session metadata in MongoDB for retrieval  

- **Privacy-Focused Design**  
  Does not store actual user content, ensuring high data privacy
